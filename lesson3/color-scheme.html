Color Scheme:

#2D2D2D is to be used for the nav bar. #DADADA is to be used for the background color.
#FFDE0C is to be used for active objects, such a link in the nav bar when hovered over.
#E3B23C is to be used as a general accent.
#AFD5AA is to be used as a general accent.
